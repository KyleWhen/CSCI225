General Information

This program takes a user's inputs through the use of the buttons on screen and allows them to add, subtract, multiply, divide, (and if we have the time to make the function we can add exponents to this sentence) their inputs. The calculator runs off of a combination of HTML plus Javascript/Jquery and is styled with CSS. If you are having trouble getting the calculator to work, please click the 'Help' button to visit the Help page which will detail how the calculator works with an explanation and accompanying videos and screenshots.

To use this calculator, press any number you see on the screen as you see fit. Once you have selected your first number(s), pick your operator sign (Plus / "+", Minus / "-", Multiply / "*", Divide / "/", or Exponent / "^"), select your next number(s), and finally hit the equals (=) button to see the end result.

If you have made a mistake with your equation at any point, just click the "C" button. This will clear your equation and allow you to input it from the very beginning.

If you are still unsure as to how the program works, please look at the accompanying screenshots and videos that will walk you through the process step-by-step.



For this example, the screenshots will be showing how to operate the calculator with addition. If you would like to use subtraction, multiplication, or division please substitute the usage of the "+" operator with your desired operator.

To start, simply select the first number you would like to start with. This example will use the number 5

Once you have your number please select an operator. This example will be using addition so you would click the "+" in this case

Once you have your first number and your desired operator please select your second number. This example will use the number 10 for its second number

Once you have your first number, your operator, and your second number, please hit the equals ("=") sign to complete the operation
